# Landing Page Design Guidelines

## Visual Identity
Inspired by Memoh's AI-centric, clean, and modern interface.

### 1. Typography
- **Primary Font:** Inter or System Sans-serif.
- **Hierarchy:**
  - H1: Bold, Large (text-4xl to 6xl), tracking-tight.
  - Body: Regular, Medium (text-base), leading-relaxed.

### 2. Color Palette (Tailwind Variable Mappings)
- **Background:** Clean white (`#FFFFFF`) or Deep Dark (`#0F172A`).
- **Primary:** Action-oriented Blue or Indigo.
- **Accents:** Modern gradients for AI features.

### 3. Layout Patterns
- **Hero Section:** Centered or Split layout with clear Value Proposition.
- **Feature Grid:** Bento-box style or clean 3-column grid.
- **Interactive Elements:** Smooth transitions using Radix Vue primitives.
- **Responsive:** Mobile-first approach using Tailwind's breakpoint system.

### 4. Component Strategy
- Use **Radix Vue** for accessible primitives (Accordions, Dialogs, Tooltips).
- Keep components atomic and reusable.
- Follow the "Clean Code" principles for Vue 3 Composition API.
